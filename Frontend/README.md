# Todo Application – Frontend (React JS)

This is the frontend part of the Todo Application built using React JS. It connects with a C# .NET Web API backend to perform CRUD operations, manage categories, priorities, and search todos efficiently.

## Features

- Create, read, update, and delete todos
- Search todos
- Filter by category (Work, Personal)
- Priority levels (High, Medium, Low)
- Responsive UI
- API integration with backend

## Tech Stack

- React JS
- JavaScript
- HTML & CSS
- Axios

## Prerequisites

Make sure you have the following installed:

- Node.js (Latest LTS recommended)
- npm (comes with Node.js)

Check installation:

```bash
node -v
npm -v

Make sure you have the following installed:

- Node.js (Latest LTS recommended)
- npm (comes with Node.js)

Check installation:

```bash
node -v
npm -v
````

## Installation

Clone the repository:

```bash
git clone <repo-url>
cd <project-folder>
```

Install dependencies:

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm run dev
```

If using Create React App:

```bash
npm start
```

## Backend Setup

Ensure the backend .NET Web API is running before starting the frontend.

Update the API base URL in your configuration file:

```js
const API_BASE_URL = "https://localhost:5001/api";
```

## Notes

* Backend must be running for API calls to work
* Enable CORS in backend
* Make sure correct API URL is configured
